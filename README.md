# BioNano-Medical-Image-ToolKit
Python toolkit for Medical Image Processing, NanoBiotechnology simulations, and Biomedical Data Analysis using Jupyter Notebooks

# BioNano AI Medical Image Toolkit
This repository contains applied Biomedical Engineering implementations focused on:
Brain MRI segmentation
ECG signal analysis
AI-based tumor detection (CNN)
Python-based biomedical simulations
Developed as part of my Master’s studies in Biomedical Engineering (Bio-Nanotechnology & Instrumentation).
